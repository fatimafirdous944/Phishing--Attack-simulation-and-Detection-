# Phishing Attack Simulation and Detection

This project demonstrates both simulation of phishing campaigns and detection techniques using machine learning.

## Features
- Phishing campaign simulation
- Phishing URL detection using machine learning
- Dataset-driven classification
- Web interface for interaction

## Technologies
- Python
- Machine Learning (Scikit-Learn)
- Flask (backend)

## Purpose
This project helps in understanding phishing attacks, how they work, and how detection systems can identify them.
